# MyMedia

## Project Description
MyMedia is a web application that allows users to organize and track their personal media collections, including movies, books, shows, and games. Users can add, edit, delete, and search items, sort them by title, type, status, rating, or notes, and toggle between dark and light mode for improved readability. The app is fully responsive and works seamlessly on both desktop and mobile devices.

## Features
- Add, edit, and delete media items
- Search and filter by title, type, status, rating, and notes
- Sort items by title, type, status, rating, or notes
- Toggle between dark and light mode

## Technologies Used
- Frontend: React, JavaScript, HTML, CSS
- Data Persistence: localStorage
- Styling: CSS Flexbox/Grid, responsive design principles
- AI Assistance: Claude Code (for coding, debugging, and iterative development)

## Setup Instructions
1. Clone the repository:
   git clone https://github.com/edwardns1/MyMedia
2. Navigate into the project folder
3. Install dependencies:
   npm install
4. Run the development server:
   npm run dev
5. Open your browser to view the app.
